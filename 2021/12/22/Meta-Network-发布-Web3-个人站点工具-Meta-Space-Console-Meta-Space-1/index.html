Meta Network 发布 Web3 个人站点工具 Meta Space Console

上线时间 12月22号 20:00 GMT+8

社交通证协议网络 Meta Network 日前发布了 Meta Space Console —— “元空间控制台”。这是一款帮助用户自行创建 Web3 个人站点、发布内容、并对内容进行管理的工具。此前参与 Meta Network 获得邀请码、并占领地块的早期用户，可以在地块界面（https://metanetwork.online ）中直接访问 Meta Space Console，来使用这一工具建立自己的元空间——Meta Space。

任何 Meta Space 用户发布的内容都将以“元数据”的方式进行分布式存储。这避免了Web2.0时代的核心弊病：平台的运营者和内容创作者始终不能明确数据的权力与责任。平台方掌控着个人数据与流量权利，面临着海量数据管理的重任；而创作者的内容和社交关系完全寄托于平台之上，受平台的统一管理，而无法真正掌控自己创造出来的内容和数据。

在 Meta Space Console，用户可以开通 Meta Space，同步以往在“瞬Matataki”发布的内容，也可以通过 Markdown 编辑器发布新的内容，以及对既有内容进行编辑，在“仪表盘”中观察内容的发布情况。这些内容在发布过程中可以随时保存草稿存证，也可以在发布时选择将元数据同步到 IPFS 及 GitHub 上，实现去中心化存储。此外，用户在开通 Meta Space 时，将会获得额外的邀请码。

关于更详尽的 Meta Space 使用指南，请参阅：https://meta-io.gitbook.io/meta-wiki/

Meta Network 旨在为创造者赋能，打造永久存储、版权记录的内容场域。目前，Meta Network 已经通过内容大厅“瞬Matataki”汇聚了上千创作者，正在逐步形成富有创造力的多元创作者生态，并已经成为全球规模最大的社交通证应用。
